ARAÇ AYIBI DAVA HAZIRLIK KONTROL LİSTESİ

Bu kontrol listesi, ayıplı ikinci el araç satışından kaynaklanan uyuşmazlıklarda dava hazırlığı için kullanılır.

1. DELİL TOPLAMA

[ ] Satış sözleşmesi / fatura / ödeme belgesi
[ ] Araç teslim tutanağı
[ ] Ekspertiz raporu (bağımsız ekspertiz firmasından)
[ ] Tamirci/servis raporu ve arıza tespit tutanağı
[ ] Araç fotoğrafları (arıza bölgeleri)
[ ] Satıcı ile yazışmalar (WhatsApp, e-posta, ihtarname)
[ ] Araç kilometre geçmişi
[ ] Sigorta hasar kaydı sorgusu (Tramer)
[ ] Noter satış sözleşmesi
[ ] Garanti belgesi (varsa)

2. HUKUKİ NİTELEME

[ ] Satıcının sıfatı belirlendi mi? (tacir / galeri / şahıs)
[ ] Alıcı tüketici mi? (TKHK uygulanabilirliği)
[ ] Ayıp türü belirlendi mi? (açık / gizli / hile)
[ ] Ayıp ihbarı süresinde yapıldı mı?
[ ] TBK 219, 223, 225, 227 uygulanabilir mi?
[ ] TKHK 8-13 uygulanabilir mi?
[ ] TTK 23 tacir satışı mı?
[ ] Zamanaşımı süresi doldu mu?

3. TALEP

[ ] Sözleşmeden dönme + bedel iadesi
[ ] Bedel indirimi (ayıp oranında)
[ ] Ücretsiz onarım bedeli
[ ] Ayıpsız misli ile değiştirme
[ ] Maddi tazminat (ekspertiz, tamir, yol masrafı vb.)
[ ] Manevi tazminat (varsa)

4. USUL

[ ] Görevli mahkeme: Asliye Hukuk / Tüketici Mahkemesi
[ ] Yetkili mahkeme
[ ] Arabuluculuk zorunlu mu? (tüketici uyuşmazlıkları için evet)
[ ] Dava değeri
[ ] Bilirkişi incelemesi talep edilecek mi?
